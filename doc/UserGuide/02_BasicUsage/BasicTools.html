Basic MinorThird Tools

Listed below are the most commonly used tools in MinorThird. For each tool a brief description of its purpose is given along the basic syntax needed to invoke it. Also, for tools that have more complex functionality a link to a tutorial is provided as well.

edu.cmu.minorthird.ui.Help

The Help tool provides a list of tools in MinorThird as well as their basic arguments. The syntax to invoke this tool is:

java edu.cmu.minorthird.ui.Help

edu.cmu.minorthird.ui.ViewLabels

The ViewLabels tool allows you graphically view a set of documents and the annotations associated with these documents. The syntax to invoke this tool is:

java edu.cmu.minorthird.ui.ViewLabels -labels <directory containing documents>

For a detailed description of the capabilities of this tool see the ViewLabels Tutorial

edu.cmu.minorthird.ui.EditLabels

The EditLabels tool allows you graphically a set of labels (annotations) associated with a set of documents. These annotations may be pre-existing (as embedded XML) or may have been created using MinorThird. One of useful feature of this tool is that it allows you to compare two different kinds of labels (called Span Types). For instance if you want to know what parts of your data an annotator has trouble with, you can graphically compare the predictions to what actually should have been labeled. The syntax to invoke this tool is:

java edu.cmu.minorthird.ui.EditLabels -labels <directory containing documents>

For a detailed description of the capabilities of this tool see the EditLabels Tutorial
